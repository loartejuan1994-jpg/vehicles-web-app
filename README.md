# vehicles-web-app

## Descripción
Aplicación web para el análisis de datos de venta de vehículos en US.

## Funcionalidad
- Visualización interactiva de la distribución del odómetro mediante un histograma.

- Análisis de la relación entre el odómetro y el precio mediante un gráfico de dispersión. 

- Casillas de verificación para activar/desactivar cada gráfico.

## Tecnologías utilizadas 

- Python
- Pandas
- Plotly
- Streamlit
